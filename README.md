# Joker

Client and web API components built in C# with .NET Core. Enjoy the jokes from https://icanhazdadjoke.com! Enjoy the 'dad jokes', *they are the funniest of all jokes*!

Start the web API:
```
$ cd JokerWebApi
$ dotnet restore
$ dotnet build
$ dotnet run
```

Run the simple console-based client:
```
$ cd JokerUi
$ dotnet restore
$ dotnet build
$ dotnet run
```
... and go nuts!
